# *정적타입 언어와 동적타입 언어의 차이*

#### 프로그래밍 언어에서 타입이란, 자료형 즉, 변수의 종류를 뜻합니다.(boolean,number,string,int...)

- 정적타입 언어(Static type language): 우선 정적타입언어의 종류는 Java,C,C#,C++,Typescript등이 있고, 이들은 변수나 함수의 타입을 컴파일 타임에 결정합니다. 다시말해, 정적타입의 언어들은 보통 컴퓨터가 직접 실행할수 있게 컴파일링 이라는 과정을 거쳐서 실행이 되게 되는데, 이 단계(컴파일 타임)에서 변수의 타입을 결정하고 에러를 발생시킵니다.

- 동적타입 언어(Dynamic type language): 우선 동적타입언어의 종류는 Javascript,R,Python,Ruby...등이 있습니다. 이들은 프로그램을 실행하는 단계에서, 즉 런타임에서 타입을 결정하고 에러를 발생시킵니다. 보통 인터프리터어 로 작동되어져서, 소스코드를 한줄한줄 인터프리터가 실행하면서 그때 타입을 결정합니다.
