# *JavaScript 타입 종류를 열거하라.*
##### JavaScript는 동적 타입 언어이지만, 크게 Primitive 타입과 Object 타입으로 나눌 수 있습니다.

## Primitive 타입은 다음과 같습니다.
#### Number: 숫자 타입
#### String: 문자열 타입
#### Boolean: 불리언 타입
#### Null: null 값
#### Undefined: undefined 값


## Object 타입은 다음과 같습니다.
#### Array: 배열 타입
#### Date: 날짜 타입
#### Function: 함수 타입
#### Object: 객체 타입
